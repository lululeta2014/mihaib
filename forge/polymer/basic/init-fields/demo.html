Test if mypropertyChanged() handlers run at initialization. Initialization can be via the element's default value or by setting an HTML attribute.
However, all examples in this page should run the *Changed() handler if you type in the field.
A simple element, with only a change handler.
No HTML attribute set: we see the default value, but the *Changed() handler never runs so Uppercase is not set.
Setting the HTML attribute runs the *Changed() handler.
An element which registers the *Changed() handler from the start but sets the field's default value in the ready() lifecycle callback, only if Object.hasOwnProperty() is false (so the user didn't set it). The field exists in the prototype because we've listed it in the polymer-element's ‘attributes’ list.
No HTML attribute set: we see the default value, and the *Changed() handler runs.
HTML attribute set, the *Changed() handler runs.
