Browser length units

You may see elements on this page at a different size than they are defined. The W3C CSS specification puts pixels and physical units (e.g. cm, inches) in a fixed ratio of 96dpi.

If your display has a different DPI (e.g. 110) you can tell Firefox about the ratio 110/96 ≈ 1.1458. Type about:config in the address bar and set layout.css.devPixelsPerPx to 1.1458.
