Pixels and DPI (dots per inch)

The W3C CSS specification puts pixels and physical units (e.g. cm, inches) in a fixed ratio “because too much existing content relies on the assumption of 96dpi”.

So the boxes below should have the same size. The first one is 3×0.5 inches, the second is 288×48 pixels.
